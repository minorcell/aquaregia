# Getting Started

## Install

```bash
cargo add aquaregia --features openai
```

## Create a client

```rust
use aquaregia::LLMClient;

let client = LLMClient::openai_compatible("https://api.deepseek.com")
    .api_key(std::env::var("DEEPSEEK_API_KEY")?)
    .build()?;
```

## Generate and stream

```rust
let out = client.generate("deepseek-chat", "hello").await?;

let mut stream = client.stream_text("deepseek-chat", "hello").await?;
while let Some(chunk) = stream.next().await {
    print!("{}", chunk?);
}
```
