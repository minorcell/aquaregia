# 迁移指南

## 核心变化

- `AiClient` -> `LLMClient`
- `generate_prompt` -> `generate` / `run`
- `stream_prompt` -> `stream` / `stream_text`
- `openai_compatible(base_url).api_key(...).build()` 为推荐初始化方式

## 迁移清单

<Steps>
### 替换类型名

统一把 `AiClient` 替换为 `LLMClient`。

### 更新方法名

`generate_prompt` 改为 `generate`；Agent 层使用 `run`。

### 工具参数类型化

将 `Value` 参数改为 `Deserialize + JsonSchema` 结构体。
</Steps>
