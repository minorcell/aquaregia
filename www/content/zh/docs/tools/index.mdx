# Tool API

Aquaregia v2 支持参数类型推导，避免手写 JSON Schema。

```rust
#[derive(Debug, serde::Deserialize, schemars::JsonSchema)]
struct WeatherArgs {
    city: String,
}

let weather = tool("get_weather")
    .description("Get weather by city")
    .execute(|args: WeatherArgs| async move {
        Ok(serde_json::json!({
            "city": args.city,
            "temp_c": 23,
        }))
    });
```
