# Aquaregia Docs

Aquaregia 是一个 provider-agnostic 的 Rust LLM SDK，聚焦三件事：

1. 易用：简单场景一行完成。
2. 可控：复杂场景提供完整扩展点。
3. 可恢复：错误类型可匹配、可重试、可降级。
